# Validation du Déploiement

## Description
Une fois l'infrastructure déployée via l'automatisation, il est nécessaire de valider que les services sont opérationnels et interagissent correctement entre eux.

## Actions Réalisées
1.  Vérification de la connectivité vers l'App Service.
2.  Vérification de la liaison entre l'App Service et la base SQL.
3.  Validation des règles de sécurité réseau (NSG).

## Livrables
*   Rapport de validation confirmant la disponibilité des services.
