# Apprentissage Manuel (ClickOps)

## Description
Cette étape consiste à créer manuellement les ressources Azure via le Portail Azure. L'objectif est de comprendre les options de configuration, les dépendances et les paramètres de chaque service avant d'automatiser leur déploiement.

## Actions Réalisées
1.  Création d'un Resource Group.
2.  Configuration d'un Virtual Network (VNet) et des Subnets.
3.  Déploiement d'une base de données Azure SQL.
4.  Déploiement d'un Azure App Service (Linux) avec un conteneur Docker.
5.  Configuration manuelle des règles de pare-feu et des chaînes de connexion.

## Livrables
*   `Azureresources.csv` : Export des ressources créées manuellement pour référence.
